# A Markdown Pull-Quote Bookmarklet
## 2012/02/20 18:06

<a href="javascript:var b=document.body;if(b&&!document.xmlVersion){void(z=document.createElement('script'));void(z.type='text/javascript');void(z.src='http://walterhiggins.net/blog/markdown_tools.js');void(b.appendChild(z));void(z=document.createElement('script'));void(z.type='text/javascript');void(z.src='http://walterhiggins.net/blog/files/markdown_pullquote.js');void(b.appendChild(z));}else{}">Markdown Pull-Quote</a>

 1. Drag the above link to your browser's toolbar to create the bookmarklet.
 2. Now when you're on a page you want to pull a quote from, select the text you want to quote, then click the bookmarklet.
 3. A Textarea will be appended to the bottom of the page you're looking at and the selected text will be shown in markdown format with a markdown style link.
 4. Press CTRL+C to copy the selected text to the clipboard and paste the text into your favorite Markdown text editor.

This hasn't been thoroughly tested yet but works on Chrome and Firefox.
